# Weather Data Analysis – Week 4 Project

## Project Overview
This project analyzes historical weather data to identify temperature trends,
monthly averages, and precipitation distribution using Python and Matplotlib.

## Objectives
- Analyze temperature trends over time
- Compare average temperature by month
- Visualize precipitation distribution
- Demonstrate a complete data analysis workflow

## Technologies Used
- Python
- Pandas
- Matplotlib

## Project Structure
- data/ – dataset used for analysis
- visualizations/ – generated charts
- report/ – written insights and conclusions

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

## Output
All charts will be saved in the `visualizations` folder.

## Data Source
The dataset used in this project was obtained from **Kaggle**.

Dataset Name: Weather History Dataset
